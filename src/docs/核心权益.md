# 核心权益 - 分配及权益

## 壹 交易税费体系
TRANSACTION TAX & FEE STRUCTURE

### 母 TOKEN 税费 (PARENT)
- **买入滑点 5%:** 2% 节点 + 3% 社区
- **卖出滑点 5%:** 2% 社区 + 3% 销毁
- **二级盈利税 10%（最高 30%）:** 底池 < 1,000 万 时卖出额外 +20%，随底池增长递减
- **每日自动递减燃烧 0.2%** 分配:
  - 回购 80%
  - 月梯 2%
  - 周梯 F%
  - 节点 1%
  - V5-V9 2%
  - 社区 14%

### 子 TOKEN 税费 (CHILD · 子符火)
- **买入滑点 5%:** 2% 节点 + 3% 社区
- **卖出滑点 5%:** 4% 销毁 + 1% 社区
- **C2C 成交税 3%:** 1% 销毁 + 2% 守护储备
- **每日自动销毁 0.1%** 分配:
  - 回购 80%
  - 月梯 2%
  - 周梯 2%
  - 节点 2%
  - V5-V9 16%（注: 可能是10%）
  - 社区 14%

### 社区收入分配:
- **40% 社区运营**
- **60% 补充国库储备池**

## 贰 五池联动权益
FIVE-POOL INTERLOCK RIGHTS

| 池名 | 代号 | 说明 | 用户入金比例 |
|------|------|------|-----------|
| 交易流动池 | TLP | 支撑母 TOKEN 交易流动性，用户入金 35% 注入 | 用户入金 35% |
| 量化执行池 | QEP | 生成 AI 量化收益，用户入金 45% 注入，收益反哺系统 | 用户入金 45% |
| 国库储备池 | TRP | 保障价格稳定与兑付，由用户入金 20% 及群体收益补充 | 用户入金 20% + 复利 |
| 子 TOKEN 底池 | CHILD POOL | 承接用户收益中的子 TOKEN（静态 18% + 动态 35%），支撑子 TOKEN 流通 | 来源: 收益承接 |
| 造血国库 | HEMATOPOIETIC TREASURY | 由 AI 智能体收益持续补充，为系统子提供长期资金支持 | 来源: AI 造血 |
