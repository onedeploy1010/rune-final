## PART I - 单币必死
Q1: 为什么价值储存与激励流通，永远不能是同一件事？
- 任何DeFi协议必须同时完成两件对立的事: 价值储存(代币要稳定升值) + 激励流通(代币要快速分发)
- 单币模型的三重死因: OHM, SLP, TIME, KLIMA
  1. 利率依赖导致: 稳定性与流动性需求互相矛盾，任何参数调节都是治标
  2. (3,3) 假博弈: 假设所有人理性，但人性贪婪，「我先跑」均衡瞬间崩溃
  3. 无外部造血: 收益依赖新入场，本质是 资金接力，数字上必然归零

## RUNE+ 的物理拆解
- RUNE + EMBER + veRUNE + VE(3,3)
  1. 功能拆分: 母币承载 价值与治理，子币承载 流通与燃料，各自优化
  2. 复合博弈: veRUNE 不可停止 + 锁定期，无法闪电贷攻击
  3. 真实造血: AI 量化外部收益注入 + 外部项目销毁，资金来自生态外

## PART II - 两枚齿轮·各司其职

### 符·RUNE / Steering Wheel (母币)
- 用户入金 35% 自动组 RUNE/USDT LP，协议永久持有，用户获得 veRUNE
- **FOUR CORE FUNCTIONS:**
  1. Epoch 投票权: 每 14 天决定 EMBER 流向哪个池，投 veRUNE 权重线上签名
  2. 期货收益权: 外部项目用真金白银的代币争夺 EMBER 流向，100% 按权重量分给 veRUNE
  3. Forge 手续费分红: Rune Pool 交易费 50% 每 Epoch USDT 形式发放
  4. DAO 治理权: 协议升级 + 参数调整 + IPO² 项目进入二次方投票

- **PRICE DRIVER:**
  Forge AI 月收益 40-50% 自动买入 RUNE → 协议 只买不卖 · veRUNE 锁仓持续拉高流通 → 数学性单向通道

### 符火·EMBER / The Fuel (子币)
- 总量 1,310 万 硬顶，通缩至 131 万(90% 销毁)，任何时刻流通 ≤ 131 万
- **FOUR CORE FUNCTIONS:**
  1. 预测市场押注: 参与 IPO² 的唯一门票，无 EMBER 不入场
  2. IPO² 打新权益: 持仓量决定打新配额，独家 EMBER 100% 销毁
  3. 期货市场约的: 外部项目竞相支付期货代币以换取 EMBER 流入自己池子
  4. AI 量化池激励: Forge Pool 不同策略竞争 EMBER 流入 → 量化收益 30% 回流 veRUNE

- **PRICE DRIVER:**
  P ∝ 有效TVL · IPO² 锁定 EMBER 100% 销毁 · 131 万生通硬顶下，流通越少 → 期货成本越高 → 死亡螺旋反向运行

## PART III - 主轴·veRUNE
THE SHAFT THAT LOCKS THE MACHINE

Q3: 为什么 veRUNE 不是装饰，而是把一切锁死的那根主轴？

- veRUNE 干一件简单事: 把控制权(投票)与价值(期货收益、手续费、空投)绑死在 RUNE 持有者身上，且让他们必须锁仓才能拿到。

## PART IV - 双飞轮·同死同生
TWIN FLYWHEELS LOCKED IN ONE AXLE

Q4: RUNE 与 EMBER，究竟是竞争还是共振？
- 既不竞争，也不简单并搞 —— 它们是 用同一根主轴(veRUNE)咬合的两个反向飞轮。
- RUNE 飞轮转动时，生成的 veRUNE 让 EMBER 流向更精确，EMBER 飞轮就转得更快
- EMBER 飞轮转动时，IPO² 活跃 → 外部项目越竞越猛 → veRUNE 越值钱 → RUNE 越被抢购
- 结果: 任何一个飞轮加速 → 主轴扭矩增大 → 另一个飞轮被强制带动。这是 ve(3,3) 把「双币」变成「永动机」的物理结构。

### 符·RUNE 飞轮 (LEFT FLYWHEEL):
1. TVL 增 → Forge AI 造血增 → 买 RUNE 多
2. RUNE 涨 → veRUNE 更值钱 → 用户选长期锁仓
3. 有效 TVL 提升 → EMBER 更稀缺 → 价涨
4. EMBER 涨 → 预测市场更活跃 → 更多项目来购期
5. 更多期货 → veRUNE 收益涨 → 更多人买 RUNE 入场
6. TVL 增 → 回到第1步，循环加速

### 符火·EMBER 飞轮 (RIGHT FLYWHEEL):
1. IPO² 活跃 → 锁定 EMBER 销毁 → 流通减少
2. EMBER 价涨 → 打新价值高 → 更多人持币参与
3. 更多持有 → 更多销毁 → EMBER 更稀缺
4. 更多外部项目想加入 IPO² → 更多期货流入
5. 更多 veRUNE 收益 → 吸引更多入场 → TVL 增
6. EMBER 更稀缺 → 回到第1步，循环加速
