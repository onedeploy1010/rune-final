## 技术方案

### 推荐动画库
1. **framer-motion** (已安装) - 主要动画引擎
2. **@xyflow/react** - 流程图可视化 (需安装)
3. **Lottie** - 高级插图动画 (可选)


## 动画步骤详细设计

### 第1步: 入单现金流 (Order Flow)
**场景**: 用户入单 → 资金分配
**动画元素**:
- [ ] 用户钱包图标 (起点)
- [ ] USDC代币飞向两个方向的动画
- [ ] 国库图标 (接收部分USDC)
- [ ] LP池图标 (接收部分USDC)
- [ ] 金额数字动态显示
- [ ] 百分比分配说明文字

**交互**: 点击后USDC从钱包分流动画

---

### 第2步: LP配对 (LP Pairing)
**场景**: 交付合约拨出B18 → 与USDC配对 → 添加流动性
**动画元素**:
- [ ] 交付合约图标
- [ ] B18代币从交付合约飞出
- [ ] USDC从LP池区域
- [ ] 两者在中间汇合形成LP代币
- [ ] LP代币进入流动性池
- [ ] AMM曲线示意图

**交互**: 点击后B18和USDC配对动画

---

### 第3步: 质押合约 (Staking Contract)
**场景**: B18代币 → 用户质押合约
**动画元素**:
- [ ] 对应数量B18代币
- [ ] 用户质押合约图标
- [ ] 质押天数选择 (360/180/90/30天)
- [ ] 每日利率显示
- [ ] 锁定倒计时动画

**交互**: 点击后B18进入质押合约动画

---

### 第4步: 每日释放 (Daily Release)
**场景**: 质押期结束 → 选择释放天数 → 计算税收
**动画元素**:
- [ ] 质押合约到期动画
- [ ] 释放天数选择 (30/15/7/1天)
- [ ] 税率对应显示 (0%/2%/4%/6%)
- [ ] 利息计算公式动画
- [ ] 税收金额分离动画

**交互**: 点击后显示释放选项和税收计算

---

### 第5步: 税收回购 (Tax Buyback)
**场景**: 税收USDC → 二级市场 → 购买B18
**动画元素**:
- [ ] 税收USDC图标
- [ ] 二级市场/AMM池图标
- [ ] B18价格上涨动画 (曲线上升)
- [ ] 购买的B18代币
- [ ] 价格变化数字动画

**交互**: 点击后USDC换成B18，价格上涨动画

---

### 第6步: SPP平衡合约 (SPP Balance)
**场景**: 税收购买的B18 → SPP合约 → 价格调控
**动画元素**:
- [ ] SPP合约图标 (盾牌/平衡秤)
- [ ] B18进入SPP动画
- [ ] 价格过高时卖出动画
- [ ] 价格过低时买入动画
- [ ] 价格稳定区间示意

**交互**: 点击后展示SPP双向调控机制

---

### 第7步: 释放提现 (Withdrawal)
**场景**: B18释放 → 国库兑付USDC → 扣税 → 分批释放
**动画元素**:
- [ ] 用户释放的B18代币
- [ ] 国库图标
- [ ] 二级市场价格参考
- [ ] 税收扣除动画
- [ ] USDC分批飞向用户钱包 (按天数)
- [ ] 进度条显示释放进度

**交互**: 点击后展示兑付和分批释放流程

---

### 第8步: 代币分配 (Token Distribution)
**场景**: 兑付的B18 → 四个去向
**动画元素**:
- [ ] 中心B18代币池
- [ ] 四个分支箭头动画
- [ ] 交付合约 (50%) - 绿色
- [ ] 销毁 (20%) - 红色火焰
- [ ] 奖励池 (20%) - 金色
- [ ] SPP (10%) - 蓝色盾牌
- [ ] 百分比数字动画

**交互**: 点击后B18分流四个方向

---

### 第9步: 433保护协议 (433 Protocol)
**场景**: 提现 > 入单 → 启动保护机制
**动画元素**:
- [ ] 天平/警报图标
- [ ] 当日入单金额 vs 当日提现金额对比
- [ ] 触发条件动画
- [ ] 三方承担动画:
  - [ ] 国库 40% (蓝色)
  - [ ] 静态奖励 30% (绿色)
  - [ ] 动态奖励 30% (紫色)
- [ ] 排队队列动画
- [ ] 恢复后倒序赎回动画

**交互**: 点击后展示433分配和排队机制

---

### 第10步: 动态贡献奖励 (Dynamic Rewards)
**场景**: 团队业绩 → 等级达标 → 级差奖励
**动画元素**:
- [ ] 团队树状结构图
- [ ] 大区/小区区分动画
- [ ] 等级阶梯图 (V1-V5等)
- [ ] 40%团队利息计算动画
- [ ] 级差制度示意
- [ ] USDC从国库兑付动画

**交互**: 点击后展示团队结构和奖励计算

---

### 第11步: 业绩奖励 (Performance Rewards)
**场景**: 奖励池B18 → 层级分配
**动画元素**:
- [ ] 奖励池B18图标
- [ ] 多层级结构图
- [ ] 第1层 100% → 逐层递减 2% → 最低10%
- [ ] 小区业绩/全网业绩计算
- [ ] B18分配到各层动画

**交互**: 点击后展示层级递减分配

## UI/UX 设计要求

### 整体风格
- [ ] 深色主题为主，适配浅色模式
- [ ] 渐变色彩 (与现有设计系统一致)
- [ ] 玻璃拟态效果 (glassmorphism)
- [ ] 流畅的曲线和圆角

### 动画要求
- [ ] 60fps 流畅动画
- [ ] 缓动函数使用 ease-out 或 spring
- [ ] 动画持续时间 300-800ms
- [ ] 支持减少动画偏好设置

### 响应式设计
- [ ] 桌面端: 横向流程图布局
- [ ] 移动端: 纵向卡片式布局
- [ ] 触摸滑动支持
- [ ] 手势导航

### 导航控制
- [ ] 步骤进度条/点指示器
- [ ] 上一步/下一步按钮
- [ ] 点击任意步骤跳转
- [ ] 键盘导航 (←/→)
- [ ] 自动播放选项
